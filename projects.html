Lab Manager

Lab Manager is built on Microsoft's ASP.NET Core framework using C#, Identity Framework and Entity Framework and SQL for data integration and management. Lab Manager seeks to fill a true need for many laboratories (environmental, healthcare or other) by providing an inventory system to track laboratory items and chemicals.

Game Closet

Inspired by a tiny closet and over 60 board games, Game Closet is a single-page application that enables board game enthusiasts to track their collection by keeping the most relevant details of a game (such as minimum and maximum players or whether the game is competitive or cooperative) at the touch of one's fingertips. This Front-End Capstone project was built with ReactJS and JSON server for persistent data storage and Reactstrap and Material Design libraries for styling.
